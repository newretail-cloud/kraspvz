# Product Backlog: Платформа для подбора персонала ПВЗ (Telegram + MAX)

## Общая информация

| Параметр | Значение |
|----------|----------|
| **Проект** | Биржа подработки для пунктов выдачи заказов (ПВЗ) |
| **Платформы** | MAX (основной), Telegram (дублирующий) |
| **Архитектура** | CORE + адаптеры, RBAC, веб-админка |
| **Монетизация** | подписки / комиссия / реклама (после 1000+ активных) |
| **MVP-сервер** | свои серверы HP Proliant G6 |
| **Переход на VPS** | при ≥1000 активных пользователей |
| **Репозиторий** | https://github.com/newretail-cloud/kraspvz |

## Легенда

- **Приоритет**: `P0` — критично (без этого нет MVP) / `P1` — важно / `P2` — желательно
- **Статус**: `📋 План` / `🚧 В работе` / `✅ Готово` / `⏸ Отложено`
- **Оценка**: человеко-часы
